📸 Image Slider Project

A simple and responsive Image Slider built using HTML, CSS, JavaScript, and Bootstrap.

🚀 Features
Next & Previous buttons
Auto-slide every 3 seconds
Responsive design
Smooth image switching
Uses Bootstrap for styling
📁 Project Structure
project-folder/
│
├── index.html     # Main HTML file
├── style.css      # Styling file
├── script.js      # JavaScript logic
└── README.md      # Project documentation
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla)
Bootstrap 5
▶️ How to Run
Download or clone the project
Open index.html in your browser

OR

Deploy using platforms like:

Vercel
Netlify
⚙️ How It Works
Images are stored in an array
Index is used to track current image
Buttons update index (next/previous)
setInterval automatically changes image every 3 seconds
🧠 Functions Used
NextImage() → shows next image
BehindImage() → shows previous image
main() → updates image source
